# Simulaciones_TFG_Guillermo_Herranz
Código usado en para simular el sistema físico considerado en el tfg "Simulación de sistemas complejos fuera del equilibrio"

En el notebook Analisis transición.nb se muestra el código usado para el análisis individual de una transición conforme aumentamos la energía

En el notebook Multiples trayectorias.nb se muestra el código usado para analizar transiciones usando condiciones iniciales distintas

En el notebook Analizar trayectorias.nb se muestra el código usado para estudiar el comportamiento del sistema para una energía fija.
